## Описание работы

Скрипт автоматически обновляет информацию о товарах на маркетплейсах Ozon и YandexMarket.

- скачивает файл с  данными об оставшихся товарах с сайта по продаже часов [timeworld.ru](https://timeworld.ru/upload/files/ostatki.zip). В файле содержится информация о количестве и ценах на часы;
- обновляет количество товаров на складе и актуализирует цены в соответствии с данными из файла.


## Цели проекта

Код написан в учебных целях — это урок в курсе по Python и веб-разработке на сайте [Devman](https://dvmn.org).
